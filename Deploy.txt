0) Стоять в папке проекта.

1) Сборка (образуется папка /build, которую нужно будет выкладывать на хост)
 npm run build

2) Для локального запуска продуктовой версии нужен пакет serve:
 npm install -g serve

3) Запуск локально продуктового сервера для кода приложения из папки /build
 serve -s build

4) Инициализация репозитория
 git init

5) Индексация изменений
 git add .

6) Начальный коммит
 git commit -m 'initial'
